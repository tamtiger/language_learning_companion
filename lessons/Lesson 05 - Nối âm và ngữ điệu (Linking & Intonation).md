# Vì sao cần Linking?

Người Việt thường nói:

```
I

need

to

check

the

API.
```

Mỗi từ tách rời nhau.

Người bản xứ thường nói:

```
I need to check the API.
```

Nghe như một dòng liên tục.

Điều này gọi là **Connected Speech**.

Nếu không biết Linking:

- Bạn sẽ nghe người bản xứ rất khó.
- Người khác sẽ cảm thấy bạn đang "đọc" chứ không phải "nói".

---

# Linking là gì?

Linking là cách nối các từ lại với nhau khi nói.

Ví dụ:

```
pick it up

↓

pickitup
```

---

```
turn it on

↓

turniton
```

---

```
check it

↓

checkit
```

---

```
call me

↓

callme
```

---

Không phải phát âm sai.

Mà là phát âm liền.

---

# Các kiểu Linking phổ biến

## 1. Phụ âm + Nguyên âm

Đây là kiểu phổ biến nhất.

Ví dụ:

```
check it

work on

turn off

pick up

log in

sign in
```

Đọc liền thay vì ngắt.

---

## 2. Nguyên âm + Nguyên âm

Ví dụ:

```
go out

I agree

do it

see it
```

Người bản xứ thường nối rất mềm.

---

## 3. Phụ âm giống nhau

Ví dụ:

```
good day

big game

last sprint
```

Không cần đọc hai lần.

Chỉ giữ một âm.

---

# Intonation là gì?

Intonation là sự thay đổi cao độ khi nói.

Không phải từ nào cũng đọc với cùng một giọng.

Ngữ điệu giúp thể hiện:

- Câu hỏi
- Câu khẳng định
- Ngạc nhiên
- Đồng ý
- Không chắc chắn

---

# Ngữ điệu xuống (Falling Intonation)

Thường dùng cho:

- Câu khẳng định
- WH Questions
- Kết thúc ý

Ví dụ:

```
The deployment was successful.

We finished the sprint.

Where is the documentation?

What happened?
```

Giọng hạ xuống ở cuối câu.

---

# Ngữ điệu lên (Rising Intonation)

Thường dùng cho:

- Yes/No Questions
- Xác nhận thông tin
- Chưa kết thúc ý

Ví dụ:

```
Are you ready?

Did you deploy it?

Can you help me?

Is the API working?
```

Giọng lên nhẹ ở cuối.

---

# Liệt kê nhiều ý

Ví dụ:

```
We need to update the API,

the database,

the documentation,

and the deployment script.
```

Giọng lên nhẹ ở các ý đầu.

Xuống ở ý cuối.

---

# Ứng dụng trong công việc

Đọc tự nhiên các câu sau:

```
Can you review my pull request?

The deployment finished successfully.

Let's discuss the architecture.

Have you checked the production logs?

I'll update the documentation later.

We're working on the payment service.
```

Tập trung vào:

- Nối âm.
- Không dừng giữa từng từ.
- Ngữ điệu cuối câu.

---

# Speaking Practice

Đọc đoạn sau như đang nói với đồng nghiệp.

> Good morning, everyone. Yesterday I finished the authentication service. Today I'm working on the payment API. I also need to review two pull requests before lunch.

Lặp lại 5 lần.

Mỗi lần:

- Tự nhiên hơn.
- Ít ngắt hơn.
- Nối âm nhiều hơn.

---

# Mini Dialogue

### A

Good morning! How's everything going?

### B

Everything's going well. I finished the deployment yesterday.

---

### A

Can you review my pull request today?

### B

Sure. I'll check it after the meeting.

---

### A

Are we deploying this afternoon?

### B

Yes, if all the tests pass.

---

Đọc theo vai.

Sau đó đổi vai.
